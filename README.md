# Focus Desk

Time Timer 스타일 포모도로 타이머와 할 일 목록을 한 화면에서 사용하는 순수 HTML/CSS/JS 앱입니다.

## 기능

- **포모도로 타이머** — 25/5/15분 프리셋, 커스텀 시간, Time Timer 스타일 빨간 웨지
- **할 일 목록** — 추가, 완료, 삭제, 필터(전체/진행중/완료)
- **더블클릭 수정** — 할 일 텍스트를 더블클릭해 바로 편집
- **드래그 정렬** — 전체 보기에서 ≡ 핸들을 드래그해 순서 변경
- **다크/라이트 모드** — 헤더 ☀/☾ 버튼으로 전환
- **배경 테마** — 웜, 오션, 포레스트, 선셋, 라벤더
- **알림** — 타이머 종료 시 브라우저 알림 + 사운드 (설정에서 끄기 가능)
- **세션 통계** — 오늘 완료 세션 수와 총 집중 시간
- **로컬 저장** — 새로고침 후에도 상태 유지

## 실행

`index.html`을 브라우저에서 열면 됩니다. 별도 빌드나 서버가 필요 없습니다.

## 공유 링크 (GitHub Pages)

`main` 브랜치에 푸시하면 GitHub Actions가 자동으로 배포합니다. 배포가 끝나면 아래 주소로 누구나 접속할 수 있습니다.

**https://jm-git-163.github.io/focus-desk/**

배포 상태는 GitHub 저장소의 **Actions** 탭에서 확인할 수 있습니다. 처음 배포이거나 Pages 설정이 없다면:

1. [focus-desk 저장소](https://github.com/jm-git-163/focus-desk) → **Settings** → **Pages**
2. **Build and deployment** → Source: **GitHub Actions**
3. `main`에 푸시 후 Actions 워크플로가 완료될 때까지 1~2분 대기

변경 사항이 바로 안 보이면 브라우저에서 **Ctrl+F5**로 새로고침하세요.

## 단축키

| 키 | 동작 |
|---|---|
| `Space` | 타이머 시작/일시정지 |
| `R` | 타이머 리셋 |
| `Enter` | 할 일 편집 저장 |
| `Esc` | 할 일 편집 취소 |

## 파일 구조

```
index.html   # 마크업
styles.css   # 스타일 + 테마
app.js       # 타이머, 투두, 설정 로직
```
